# Sign Up for a GitHub Copilot License

This guide walks you through the process of signing up for a GitHub Copilot license for this workshop. Follow each step carefully to ensure you have full access to GitHub Copilot in VS Code.


## Prerequisites

- A GitHub account
- [Visual Studio Code](https://code.visualstudio.com/) installed on your machine
- Access to the email associated with your GitHub account


## Step 1: Fill Out the Sign-Up Form

1. Open the sign-up form: **[https://forms.cloud.microsoft/r/FDAeQ8byEW](https://forms.cloud.microsoft/r/FDAeQ8byEW)**
2. Complete all required fields in the form.
3. Make sure you provide the **GitHub username** you intend to use during the workshop.
4. Submit the form.

> **Note:** After submitting the form, you will receive an invitation via email. Please allow a few minutes for the invitation to arrive.


## Step 2: Accept the GitHub Organization Invitation

1. Check the **email associated with your GitHub account** for an invitation to join the workshop organization.
2. Check the box **Ask for a Copilot seat**
   ![join organization](.labs/lab3/resources/img/join-organization.png)
4. Wait for the seat to be provisioned — this may take a few moments.



## Step 3: Restart VS Code and Log In

1. Restart Visual Studio Code
2. Log in with the GitHub username that you used in the sign-up form.
3. If you are already signed in, you may need to sign out and sign back in for the new Copilot license to take effect:
   - Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`)
   - Type GitHub Copilot: Sign Out and select it
   - Then sign back in with your GitHub account


## Step 4: Verify Your Setup

After signing in, verify that GitHub Copilot is active:

1. You should see the Copilot icon in the VS Code status bar (bottom of the window).
2. Open the Copilot Chat panel — you should see available the follwoing models and premium


## Troubleshooting

| Issue | Solution |
|---|---|
| Didn't receive the invitation email | Check your spam/junk folder. Make sure the email on your GitHub account is correct. |
| Can't find the option to request a Copilot seat | Ensure you've accepted the org invitation first. Try refreshing the page. |
| Copilot not showing in VS Code | Restart VS Code completely. Make sure the GitHub Copilot extension is installed. |
| Signed in but no models/premium | Sign out and sign back in. Ensure your Copilot seat has been provisioned. |


## Next Steps

Once your Copilot license is active and verified, proceed to **[Lab 1](.labs/lab1/lab.md)** to begin the workshop!
